稀土金屬提煉與純化基本設計 (BEDP) 專案指南

Rare Earth Metal Refining & Purification Basic Engineering Design Package Guide

本文件旨在協助制定基本設計包 (BEDP) 委外發包書 (RFP)。內容包含標準設計產出清單、投標廠商資格審查標準，以及啟動設計前需準備的基礎資料清單。
This document is designed to assist in preparing the Request for Proposal (RFP) for the Basic Engineering Design Package (BEDP). It includes standard deliverables, vendor qualification criteria, and a checklist of basic data required before initiating the design.

1. 製程概念設計 (Conceptual Design) 產出項目清單

此階段為我司協助進行的前期工程，旨在驗證技術路線與商業可行性，為後續 BEDP 奠定堅實基礎。

中文項目名稱 English Deliverable 說明 / Description
1. 技術路線評估與選擇 Technology Route Selection 針對不同純化方法（如萃取劑種類、沉澱法）進行優劣勢與經濟性評估。
2. 方塊流程圖 Block Flow Diagram (BFD) 呈現主要化學反應與物理分離單元的處理順序及物流大方向。
3. 初步物料平衡表 Preliminary Mass Balance 計算關鍵原料消耗、主要產品產出與廢棄物初步估計量。
4. 主要設備清單與初估 High-level Equipment List 列出核心反應器與分離槽體，並初步概算其體積或處理量。
5. 資本與營運成本概估 CAPEX & OPEX Estimate 提供數量級估算 (Order of Magnitude, 精度 +/- 30~50%)，供客戶進行內部財務評估。

2. 基本設計包 (BEDP) 產出項目清單 / Deliverables of BEDP

要求工程公司於基本設計階段完成並交付的標準技術文件清單，用於銜接後續 FEED 與 EPC。

中文項目名稱 English Deliverable 說明 / Description
1. 設計基準與製程說明 Design Basis & Process Description 整體專案的產能、原料規格、產品規格，以及化學反應與製程原理的文字說明。
2. 製程流程圖 (PFD) Process Flow Diagram (PFD) 顯示主要設備、物流走向、操作溫度與壓力的核心流程圖。
3. 物料與熱量平衡表 (H&MB) Heat and Material Balance (H&MB) 各物流節點的流量、組成成分、相態、溫度、壓力及熱焓等數據 (通常來自模擬軟體)。
4. 管線與儀表流程圖 (P&ID) Piping & Instrumentation Diagram (P&ID) 包含所有設備、管線尺寸、閥門、儀表及控制迴路的詳細工程圖面 (基本設計階段發行版)。
5. 主要設備數據表/規格書 Equipment Data Sheets 反應器、萃取槽、分離塔、熱交換器、泵浦等設備的尺寸、材質(針對稀土腐蝕性特殊考量)、操作與設計條件。
6. 儀表與控制邏輯說明 Process Control Philosophy & Interlock List 描述工廠自動化控制邏輯、安全連鎖保護機制 (Cause & Effect Matrix)。
7. 公用設施消耗清單 Utility Consumption Summary 正常操作與最大負載下，對水、電、蒸汽、冷卻水、儀表空氣的需求量彙整。
8. 廢棄物與排放物清單 Effluent & Emission Summary 固廢 (如尾礦、廢渣)、廢水、廢氣的排放量與預期成分，供後續環保設計使用。
9. 初步設備佈置圖 Preliminary Plot Plan 依據設備尺寸與製程重力流/安全間距要求，規劃的初步廠區與車間佈置。
10. 危害性與可操作性分析報告 HAZOP Report (Preliminary) 基於 P&ID 進行的初步工安與風險審查會議紀錄及改善建議清單。

3. 工程公司執行能力審核清單 (發包評估標準) / Capability Evaluation Criteria

寫入 RFP 中的投標廠商資格要求，確保得標者具備將實驗室配方轉化為商業化建廠設計的能力。

評估構面 / Category 具體審核要求與證明文件 / Specific Requirements & Proof
1. 化工製程電腦模擬能力
Process Simulation Capability
  • 要求：具備使用 Aspen Plus, PRO/II 或 ChemCAD 等專業軟體進行熱量與物料平衡模擬的能力。針對稀土製程，需具備電解質(Electrolyte)或濕法冶金(Hydrometallurgy)模組的使用經驗。
  • 要求證明：提供過去執行類似複雜流體或特殊化學反應的模擬報告範例，或模擬軟體原廠授權與工程師認證。
2. 製程放大與商業化經驗
Scale-up & Commercialization
  • 要求：具備將客戶「實驗室數據 (Lab-scale)」或「模試廠數據 (Pilot-scale)」放大設計為「商業化量產廠 (Commercial-scale)」的實績。
  • 要求證明：列舉至少兩項從授權方(Licensor)或客戶取得技術轉移，並成功產出工藝包的專案實績表。
3. 特殊材質與防腐蝕設計能力
Material Selection Expertise
  • 要求：稀土提煉常涉及強酸、強鹼及高腐蝕性溶劑(如HCl, H2SO4, 萃取劑)。廠商需有專職材料工程師，能正確選用特種合金、襯氟(PTFE/PFA)或玻璃內襯設備。
  • 要求證明：提供過去處理高腐蝕性流體建廠專案的設備清單或材料選擇報告 (Material Selection Report) 經驗。
4. FEED 與 EPC 銜接經驗
FEED & EPC Interface Experience
  • 要求：產出的 BEDP 必須符合國際工程標準，確保能無縫移交給後續的細部設計(Detailed Engineering)或統包商(EPC)進行施工。
  • 要求證明：提供其標準工程圖框、P&ID 圖例標準 (Legend & Symbols) 範本，及過去協同 EPC 廠商完成建廠的成功案例。

專案啟動資料需求 (Client's Design Input)

為了讓工程公司能夠順利開展模擬與基本設計，客戶端（製程開發者）必須提供完整且可靠的實驗與配方數據。以下清單可作為客戶內部資料整理的 Check-list。

1. 目標基準 (Target Basis)

1.1 目標產能 / Target Capacity

預計年產量 (例如：噸/年) 以及設計的年操作時數 (通常為 8,000 小時/年)。

1.2 原料規格 / Raw Material Specifications

稀土礦源或前驅物的詳細化學成分、雜質含量上限、粒度分佈、含水率等。

1.3 產品規格 / Product Specifications

最終稀土產品的純度要求 (例如 99.99%)、容許雜質清單、物理型態 (粉末、結晶等)。

2. 化學與熱力學數據 (Chemistry & Thermodynamics)

2.1 反應方程式 / Chemical Reactions

主反應與已知副反應的化學方程式，以及反應熱 (放熱或吸熱數據)。

2.2 轉換率與收率 / Conversion & Yield

實驗室測得的單步驟反應轉換率、各分離/萃取步驟的目標金屬回收率 (Recovery rate)。

2.3 特殊物性數據 / Physical Properties

若使用特殊萃取劑或非標準溶劑，需提供其密度、黏度、比熱、沸點與蒸汽壓等數據，供模擬建置使用。

3. 操作條件與流程 (Operating Conditions)

3.1 實驗室方塊流程圖 / Block Flow Diagram (BFD)

呈現主要處理步驟 (如：浸出 Leaching -> 萃取 Extraction -> 沉澱 Precipitation -> 焙燒 Roasting) 的順序。

3.2 關鍵操作參數 / Key Operating Parameters

各步驟的最佳操作溫度、壓力範圍、反應停留時間 (Residence time)、pH 值控制要求及攪拌轉速要求。

3.3 相分離數據 / Phase Separation Data

如液-液萃取中的分相時間 (Settling time)、固-液分離中的過濾速率或沉降速率。

4. 環安衛與廠址條件 (EHS & Site Conditions)

4.1 危害物質清單 / Hazardous Materials Data

製程中涉及化學品的安全資料表 (SDS)，特別標註其毒性、易燃性及高腐蝕性限制。

4.2 特殊材質經驗 / Material Experience

在實驗室或模試階段發現的不適用材質 (例如：某種溶液會嚴重腐蝕 316L 不鏽鋼)。

4.3 廠址公用設施條件 / Site Utility Available

預定建廠地點可提供的公用設施規格 (如：冷卻水溫度、蒸汽壓力等級、電力電壓頻率等)。若尚未選址則由工程公司假設。

常見迷思：實驗室能做出來，就能直接發包蓋廠？

許多專業的研發人員常誤以為只要在實驗室取得良好的轉換率或純度，就能直接找營造廠或設備商蓋廠。實際上，從「燒杯 (Beaker)」放大到「商業化槽體 (Commercial Tank)」，會面臨熱傳遞、質傳流體力學、連續操作穩定性及設備腐蝕等巨大挑戰。

因此，概念設計 (Conceptual Design)、基本設計 (BEDP) 與前端工程設計 (FEED) 是將「實驗室配方」安全且具經濟效益地轉譯為「工程藍圖」不可省略的關鍵橋樑。跳過這些步驟直接進行統包 (EPC)，往往導致嚴重的設計變更、預算超支或甚至試車失敗。

製程開發到建廠生命週期 / Project Lifecycle

基準估算：年產 1,000 噸稀土金屬試驗/小量產廠（全新製程開發）

12~24 個月

1. 產品與製程開發 (Lab & Pilot) (12~24 個月)

執行者：研發團隊 (R&D)

目標： 確立全新化學反應途徑、找出最佳配方、驗證產品純度。通常在批次(Batch)實驗室或小型模試廠進行。

產出： 基礎化學與熱力學數據、單元操作的初步可行性。

3~4 個月

2. 製程概念設計 (Process Conceptual Design) (3~4 個月)

執行者：我司顧問團隊 / 製程工程師

目標： 評估商業化放大的可行性與經濟性。進行初步的製程模擬，選擇最適合的分離與純化技術路線。因屬全新製程，此階段需與研發團隊密切且反覆地進行討論與驗證。

產出： 方塊流程圖 (BFD)、初步物料平衡、概估投資成本 (Order of Magnitude Estimate)。

4~6 個月

3. 基本設計 (Basic Engineering, BEDP) ★ 核心樞紐 (4~6 個月)

執行者：我司專業製程設計團隊

目標： 將概念轉化為具體的工程參數。完成詳細的軟體模擬，決定所有設備的尺寸、材質及管線流體條件，並確立安全控制邏輯。

產出： PFD、P&ID、設備規格書、H&MB (如本網頁第一部分所述)。
※ 這是將「Know-how」定型為「工程文件」的最關鍵步驟。

2~3 個月

4. 前端工程設計 (FEED) (2~3 個月)

執行者：專業工程公司 (具備工程顧問 Engineering Consultant 能力)

目標： 基於 BEDP 進一步深化設計，通常為了準確估算建廠總成本 (精度 +/- 10~15%) 並制定統包發包文件。此階段會決定長交期設備 (Long-lead items) 並進行初步 3D 佈置。

產出： EPC 發包書 (RFP for EPC)、預算估價單、細化的 P&ID。

10~14 個月

5. 細部設計、採購與建造 (EPC) (10~14 個月)

執行者：專業工程公司 (具備統包 EPC 能力)

目標： 執行管線等距圖 (Isometric drawings)、土木結構計算、電氣拉線等細部設計。採購所有實體設備與材料，並進駐現場施工與安裝。

產出： 一座實體完工的稀土提煉廠 (Mechanical Completion)。

2~3 個月

6. 試車與投產 (Commissioning & Start-up) (2~3 個月)

執行者：我司專業製程設計團隊 (主導) + 客戶操作團隊 (學習) + EPC (機械試車支援)

目標： EPC 廠商通常僅負責前期的機械與水試車 (Mechanical/Water run)。後續核心的化學試車 (Chemical run) 與正式投料，必須由最了解製程條件的我司設計團隊來主導參數調整。客戶團隊需全程參與操作學習以利日後接手。最終驗證產能與純度是否達到 BEDP 設定的性能保證 (Performance Guarantee)。

產出： 順利移交營運 (Handover to Operation)。
